# 🎬 Netflix Analytics Dashboard

An interactive **Netflix Analytics Dashboard** built using **Power BI**, **SQL**, and **Microsoft Excel** to analyze Netflix's content library and uncover key business insights.

## 📊 Features
- Movies vs TV Shows analysis
- Genre and rating distribution
- Country-wise content analysis
- Release year trends
- Interactive filters and slicers

## 🛠️ Tools Used
- Power BI
- SQL
- Microsoft Excel

## 📂 Dataset
Netflix Titles Dataset

## 🚀 Outcome
Developed an interactive dashboard that transforms raw Netflix data into meaningful visual insights for better decision-making.
